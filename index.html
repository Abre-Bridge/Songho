SONGO BRIDGE

Le jeu de pions traditionnel du Cameroun

Règles

  1. Semez dans le sens horaire depuis une case contenant au moins 1 graine.
  2. Capture : si la dernière graine tombe dans une case adverse contenant 2 à 4 graines, vous les capturez.
  3. Capture en série : capturez aussi toutes les cases adjacentes consécutives dans le territoire adverse.
  4. Capture case 1 : si le nombre de graines semées est un multiple de 14 et tombe sur la case adverse 1, une graine est capturée.
  5. Case 7 interdite : si la case 7 contient 1 ou 2 graines, l'adversaire les reçoit.
  6. Solidarité : si l'adversaire n'a plus de graines, vous devez jouer la case ayant le plus de graines.
  7. Fin du jeu : le premier à 40 graines gagne.
